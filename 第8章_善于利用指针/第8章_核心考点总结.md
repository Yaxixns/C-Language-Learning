# 第8章 善于利用指针 — 核心考点总结

---

## 一、本章核心知识点

### 1.1 指针基本概念（⭐必考）

```c
int a = 10;
int *p;        // 定义指针变量
p = &a;        // p 指向 a（存 a 的地址）
*p = 20;       // 通过指针修改 a 的值
```

| 符号 | 含义 | 示例 |
|------|------|------|
| `&` | 取地址 | `&a` → a 的地址 |
| `*` | ①定义指针 ②间接访问 | `int *p;` / `*p = 5;` |
| `p` | 指针变量本身（存的地址） | `p = &a` |

### 1.2 指针与数组（⭐高频）

```c
int a[5] = {1, 2, 3, 4, 5};
int *p = a;      // p 指向 a[0]，等价于 p = &a[0]

// 以下等价：
a[i]  ⇔  *(a + i)
&a[i] ⇔  a + i
```

| 要点 | 说明 |
|------|------|
| 数组名 = 首地址常量 | `a` 等价 `&a[0]`，但 `a++` 非法 |
| 指针可自增 | `p++` 指向下一个元素（自动跳过 sizeof 字节） |
| `*p++` | 先取 `*p`，再 `p++` |
| `(*p)++` | 将 `*p` 的值加1 |

### 1.3 指针与字符串

```c
char str[] = "Hello";      // 字符数组
char *p = "Hello";         // 指针指向字符串常量（不可修改）
char *name[] = {"Jan", "Feb", "Mar"};  // 指针数组
```

| 对比 | `char s[]` | `char *p` |
|------|------|------|
| 可否修改内容 | ✅ 可修改 | ❌ 不可修改（常量区） |
| 可否重新指向 | ❌ s 是常量 | ✅ p = 新地址 |

### 1.4 指针与函数（⭐重点）

#### 指针作参数 → 能修改实参
```c
void swap(int *a, int *b)   // 传地址
{
    int t = *a;
    *a = *b;
    *b = t;
}
// 调用：swap(&x, &y);  此时 x,y 能被真正交换
```

#### 函数返回指针
```c
int *func() { ... }   // 返回指针类型
```

#### 函数指针
```c
int (*p)(int, int);    // 指向返回int、带两个int参数的函数
p = max;               // p 指向 max 函数
(*p)(a, b);            // 通过指针调用
```

### 1.5 指针的指针（二级指针）

```c
int a = 10;
int *p = &a;
int **pp = &p;    // pp → p → a
// **pp = 20;     // 修改 a
```

### 1.6 动态内存分配

```c
void *malloc(size_t size);   // 分配内存，返回 void*
void free(void *p);          // 释放内存
```

## 二、高频考点（期末必考）

### 🔥 考点1：指针基本运算
```c
int a=5, *p=&a;
*p = 10;    // a 变成 10
(*p)++;     // a 变成 11
*p++;       // 先取*p，然后 p 指针后移（危险！）
```

### 🔥 考点2：指针与数组混用
- `a[i]`=`*(a+i)`=`p[i]`=`*(p+i)` 全部等价
- 指针遍历数组：`for(p=a; p<a+n; p++)`

### 🔥 考点3：指针作函数参数
- 传普通变量 → 不能修改实参（值传递）
- 传指针 → 能修改实参指向的值
- 传数组 → 数组名即地址，能修改

### 🔥 考点4：指针数组
```c
char *name[] = {"Tom", "Jerry", "Spike"};
// name[0] 指向 "Tom"，name[1] 指向 "Jerry"
```

## 三、易错点警示（⚠️ 坑点汇总）

| 易错点 | 说明 |
|--------|------|
| 野指针 | `int *p; *p=5;` 未初始化就使用 → 崩溃 |
| `*p++` vs `(*p)++` | 前者移动指针，后者修改值 |
| 指针字符串不可改 | `char *p="Hello"; p[0]='h';` → 崩溃 |
| 数组名不是变量 | `a++;` 错误，数组名是地址常量 |
| 指针越界 | 指针访问超出数组范围的地址 |
| 返回局部变量地址 | `return &a;` 函数结束后 a 已销毁 |
