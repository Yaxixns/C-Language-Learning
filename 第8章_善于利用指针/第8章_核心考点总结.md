## 一、指针核心概念

### 1.1 指针即地址

```c
int a = 10;
int *p = &a;      // p 存的是 a 的地址，*p 是 a 的值
printf("%d", *p); // 10 — 解引用
```

### 1.2 指针层级

| 写法 | 含义 | 示例 |
|------|------|------|
| `int *p` | 指向 int 的指针 | `p = &a;` |
| `int **pp` | 指向指针的指针（二级） | `pp = &p;` |
| `int (*p)[N]` | 指向数组的指针 | 二维数组参数 |
| `int *p[N]` | 指针数组 | `char *months[12]` |
| `double (*f)(double)` | 函数指针 | 传 `sin`/`cos`/`exp` |

---

## 二、指针运算

```c
int a[] = {1,2,3,4,5};
int *p = a;          // 或 int *p = &a[0];

// 四种等价写法访问 a[i]：
a[i]  ==  *(a + i)  ==  p[i]  ==  *(p + i)

// p++ 移向下一个元素（自动跳过 sizeof(int) 字节）
// 二维：a[i][j] == *(*(a + i) + j)
```

## 三、本章错题复盘

### ⚠️ 8-6 字符串长度 — 未按要求封装
**错误**：把求长度逻辑写在 main 里，题目要求"写一函数"  
**修正**：提取 `int str_len(char *p)` 函数

### 🔥 8-15 第1门课平均分 — 指针解引用错误
**错误**：`sum += **(score)` — `**(score)` 等价于 `score[0][0]`，循环4次加了同一个值  
**修正**：`sum += *(*(score + i))` — 等价于 `score[i][0]`，取每个学生的第1门课

### ⚠️ 8-18 月份英文名 — 缺输入校验
**错误**：`scanf("%d",&n); puts(*(months+n-1))` — n=13 时越界  
**修正**：`if(n>=1 && n<=12) puts(...); else printf("输入错误");`

## 四、指针经典用法速查

| 题型 | 形参写法 | 核心技巧 |
|------|---------|---------|
| 指向普通变量 | `int *p1, *p2` | swap 交换 `*p1` 和 `*p2` |
| 指向一维数组 | `int *arr` | `*(arr+i)` 等价 `arr[i]` |
| 指向二维数组 | `int (*a)[5]` | `*(*(a+i)+j)` 等价 `a[i][j]` |
| 字符串操作 | `char *p` | `while(*p) ...`，遇 `\0` 停止 |
| 指针数组(不等长串) | `char *p[]` | 排序只交换指针指向，不挪字符串 |
| 二级指针 | `int **p` 或 `char **p` | `**(p+i)` 比较值，交换 `*(p+i)` 指针 |
| 函数指针 | `double (*f)(double)` | 传函数名作参数 |
| 动态内存 | `malloc` / `free` | `int *a = malloc(n*sizeof(int));` |

```c

// 1=存活 0=淘汰，i=(i+1)%n 循环，count到3淘汰
free(a);  // 别忘释放
```


### 提取整数（8-16）
```c
if (*(str+i) >= '0' && *(str+i) <= '9') {
    temp = temp * 10 + (*(str+i) - '0');    // 累积数字

    temp = 0; in_num = 0;
}
// ⚠️ 别忘了字符串末尾可能以数字结尾，出循环再检查一次 in_num
```

## 六、本章易错清单

1. 指针没初始化就解引用 → 野指针，程序崩溃
2. `scanf("%d", p)` 不需要 `&`（p 本身就是地址）
3. `int *p` 的 `*` 和 `*p` 的 `*` 含义不同（定义 vs 解引用）
4. `*(a + i)` 和 `*a + i` 完全不同！前者是 `a[i]`，后者是 `a[0]+i`
5. 指针数组排序记得只交换指针，不交换字符串本体
6. `malloc` 后必须 `free`，否则内存泄漏
7. 二级指针 `**p` 做参数时，`sizeof` 和数组维度容易搞混
8. 数组名是指针常量：`a = p` 非法，`p = a` 合法
