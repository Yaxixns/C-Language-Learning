# C语言程序设计

> 基于谭浩强《C程序设计（第五版）》的课后习题全解 + 期末复习资料

## 项目概览

- **教材**：《C程序设计（第五版）》谭浩强
- **习题覆盖**：第1~8章全部课后习题（107 道 C 源文件）
- **复习资料**：8 章核心考点总结 + 期末总复习手册 + PTA 练习题合集

## 目录结构

```
├── 第1章_程序设计和C语言/          # 概念题 + Hello World
├── 第2章_算法——程序的灵魂/          # 流程图 / N-S图 / 伪代码
├── 第3章_顺序程序设计/              # printf / scanf / 变量类型
├── 第4章_选择结构程序设计/          # if-else / switch / 逻辑运算
├── 第5章_循环结构程序设计/          # while / do-while / for / 迭代
├── 第6章_利用数组处理批量数据/      # 数组 / 排序 / 字符串 / 矩阵
├── 第7章_用函数实现模块化程序设计/  # 函数 / 递归 / 函数指针 / 结构体
├── 第8章_善于利用指针/              # 指针 / 二级指针 / 动态内存
├── C语言期末总复习手册.md           # 📘 全书考点 + 错题复盘 + 输入输出速查
├── PTA练习题合集.md                 # 📋 课堂 PTA 作业汇总
└── README.md
```

每章目录下直接放置 `.c` 源码、`.exe` 编译产物和 `第X章_核心考点总结.md`。

## 快速开始

### 编译运行单个习题

```bash
gcc 第3章_顺序程序设计/3-6.c -o 3-6.exe
./3-6.exe
```

### 复习路线建议

| 阶段 | 内容 | 建议时间 |
|------|------|---------|
| 基础三章 | 第3章(输入输出) → 第4章(选择) → 第5章(循环) | 先过 |
| 重点三章 | 第6章(数组) → 第7章(函数) → 第8章(指针) | 重点攻克 |
| 总复习 | 阅读 `C语言期末总复习手册.md` | 考前集中看 |
| PTA 回顾 | 阅读 `PTA练习题合集.md` | 临考刷一遍 |

## 关键考点速览

| 章节 | 重点 | 高频坑 |
|------|------|--------|
| 第3章 | printf 格式符、scanf 必须加 `&` | `%c` 不跳过空白、char 溢出 |
| 第4章 | if-else 配对、switch、短路求值 | `=` vs `==`、case 穿透 |
| 第5章 | 三种循环选择、break vs continue | `do-while` 分号、整除 `4/3=1` |
| 第6章 | 数组排序、字符串 `\0`、折半查找 | 下标越界、`scanf("%s")` 遇空格停 |
| 第7章 | 函数定义/声明、递归、函数指针 | 局部数组返回、全局变量滥用 |
| 第8章 | `&` 和 `*`、指针数组/数组指针、动态内存 | 野指针、`malloc` 忘 `free`、`p++` vs `(*p)++` |
