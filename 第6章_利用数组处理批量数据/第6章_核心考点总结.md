## 一、数组核心知识

### 1.1 一维数组

```c
int a[5] = {1,2,3,4,5};     // 全初始化
int a[5] = {1,2};            // 部分初始化 → {1,2,0,0,0}
int a[] = {1,2,3};           // 长度自动为3
int a[5] = {0};              // 全部置0
// ⚠️ 下标从0开始：a[0] 到 a[4]
```

### 1.2 二维数组

```c
int a[3][3] = {{1,2,3},{4,5,6},{7,8,9}};
int a[][3] = {{1,2,3},{4,5,6}};       // 第一维可省
```

### 1.3 字符数组

```c
char s[] = "hello";          // 自动加\0，占6字节
char s[5] = {'h','e'};       // 不是字符串！缺\0

gets(s);      // 读到换行，丢弃\n（❌不安全）
fgets(s,50,stdin); // 安全，但保留\n
scanf("%s",s); // 遇空白停止
puts(s);      // 输出并自动换行
```

## 三、经典算法速查

| 算法 | 核心代码 |
|------|---------|
| 筛选法素数 | `for(i=2;i*i<=100;i++) if(a[i]) for(j=i*i;j<=100;j+=i) a[j]=0;` |
| 选择排序 | `min=i; for(j=i+1;j<n;j++) if(a[j]<a[min])min=j; swap(a[i],a[min]);` |
| 冒泡排序 | `for(i=0;i<n-1;i++) for(j=0;j<n-1-i;j++) if(a[j]>a[j+1])swap;` |
| 有序插入 | 找位→后移→插入。边界：比所有大时直接放末尾 |
| 逆序 | `for(i=0;i<n/2;i++) swap(a[i],a[n-1-i]);` |
| 折半查找 | 升序: `a[m]<x→l=m+1`；降序: `a[m]>x→l=m+1` |
| 杨辉三角 | `a[i][0]=a[i][i]=1; a[i][j]=a[i-1][j-1]+a[i-1][j];` |
| 鞍点 | 行最大→该列最小→判相等。同行多个最大值可能漏检 |
| 魔方阵 | 1放首行中。右上为空→去右上，被占→去正下方 |
| 密码翻译 | `'A'+'Z'-c` = 反向字母 (Atbash) |

## 四、手写字符串函数

| 函数 | 核心 |
|------|------|
| strlen | `while(*p)i++,p++;` |
| strcpy | `while(*s2) *s1++=*s2++; *s1='\0';` |
| strcat | 先找到s1的\0，再copy s2 |
| strcmp | 逐字比较，遇不等返回差值；都到\0返回0 |

## 五、易错清单

1. 下标越界不报错，运行时悄悄出问题
2. `scanf("%s")` 遇空格就停
3. 字符串忘加 `\0` → 输出后面跟乱码
4. `gets()` 丢弃 `\n`，`fgets()` 保留 `\n`
5. 二维数组作参数：`int a[][N]` 第二维必须写
6. 折半查找数组必须先排序
