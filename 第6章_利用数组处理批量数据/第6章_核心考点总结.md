# 第6章 利用数组处理批量数据 — 核心考点总结

---

## 一、本章核心知识点

### 1.1 一维数组（⭐必考）

#### 定义与初始化
```c
int a[10];                        // 定义（未初始化，值随机）
int a[5] = {1, 2, 3, 4, 5};      // 全部初始化
int a[5] = {1, 2};                // 部分初始化，其余自动为0
int a[] = {1, 2, 3};              // 不指定长度，自动为3
```

| 要点 | 说明 |
|------|------|
| 下标从**0**开始 | `a[0]` 是第一个元素，`a[9]` 是第十个 |
| 越界不报错 | C不检查下标越界，但会造成严重bug |
| 数组名 = 首地址 | `a` 等价于 `&a[0]` |
| 长度必须为常量 | `int n=10; int a[n];` 在C89中非法 |

#### 经典操作
```c
// 遍历
for (i = 0; i < n; i++) { ... }

// 逆序
for (i = 0; i < n/2; i++)
    { t = a[i]; a[i] = a[n-1-i]; a[n-1-i] = t; }
```

### 1.2 二维数组

#### 定义与初始化
```c
int a[3][4];                             // 3行4列
int a[3][4] = {{1,2,3,4}, {5,6,7,8}, ...}; // 按行初始化
int a[][3] = {{1,2,3}, {4,5,6}};        // 可不指定行数，列数必须指定
```

- 存储：**按行存放**，先行后列
- `a[i][j]` 访问第 i 行第 j 列

### 1.3 字符数组与字符串（⭐重点）

```c
char str[10] = {'H','e','l','l','o'};     // 字符数组
char str[10] = "Hello";                    // 字符串（自动加'\0'）
char str[] = "Hello";                      // 长度为6（5个字符+'\0'）
```

| 关键概念 | 说明 |
|------|------|
| 字符串结束标志 | `'\0'`（ASCII码0），不显示 |
| `"a"` vs `'a'` | `"a"` 占2字节（含`\0`），`'a'` 占1字节 |
| `gets()` | 读入一行字符串（遇回车结束，自动加`\0`），不会越界检查 |
| `puts()` | 输出字符串并自动换行 |
| `strlen()` | 求字符串长度（不计`\0`） |

### 1.4 常用字符串函数（`<string.h>`）（⭐高频）

| 函数 | 作用 | 关键点 |
|------|------|--------|
| `strlen(s)` | 求长度 | 不计`\0` |
| `strcpy(s1, s2)` | 复制 s2→s1 | s1 必须足够大 |
| `strcat(s1, s2)` | 连接 s2 到 s1 末尾 | s1 必须足够大 |
| `strcmp(s1, s2)` | 比较 | 返回值：0相等 / 正数 s1>s2 / 负数 s1<s2 |

#### 🔍 四大函数详解

**① `strlen` — 求长度**

```c
size_t strlen(const char *s);
```

- 返回 `\0` **之前**的字符个数，**不含** `\0`
- ⚠️ **常考陷阱**：`char a[] = "abc";` → `strlen(a)` 返回 **3**，但 `sizeof(a)` 返回 **4**（带了 `\0`）

**② `strcpy` — 字符串拷贝**

```c
char *strcpy(char *dest, const char *src);
```

- 把 `src` **包括 `\0`** 一起拷到 `dest`
- ⚠️ **dest 必须足够大**，否则越界——最爱考的坑
- ⚠️ **不能拷给字符串常量**：`char *s = "abc"; strcpy(s, "xyz");` → 炸，常量区不可写

**③ `strcat` — 字符串连接**

```c
char *strcat(char *dest, const char *src);
```

- 从 `dest` 的 `\0` 位置开始覆盖写入 `src`，最后补 `\0`
- ⚠️ **dest 容量 ≥ strlen(dest) + strlen(src) + 1**
- ⚠️ **dest 不能是字符串常量**，原因同 strcpy

```c
// 示例
char a[20] = "Hello";
char b[] = " World";
strcat(a, b);
printf("%s", a);   // 输出: Hello World
```

> 📝 **手动实现 strcat（理解底层原理）**：
> ```c
> char *mystrcat(char *dest, const char *src)
> {
>     char *p = dest;
>     while (*p != '\0')    // ① 走到 dest 末尾
>         p++;
>     while (*src != '\0')  // ② 逐字符拷贝 src
>         *p++ = *src++;
>     *p = '\0';            // ③ 补上结束符
>     return dest;
> }
> ```
> 三步：**走到尾巴 → 逐字拷贝 → 补 `\0`**

**④ `strcmp` — 字符串比较**

```c
int strcmp(const char *s1, const char *s2);
```

| 返回值 | 含义 |
|--------|------|
| 0 | s1 == s2 |
| 正数 | s1 > s2（字典序） |
| 负数 | s1 < s2 |

- ⚠️ **不能用 `==` 比较字符串**：`if (s1 == s2)` 比的是地址，不是内容！
- 比较规则：逐字符比 ASCII，遇到不同或 `\0` 停止
- ⚠️ 返回值不是固定的 ±1，是 **ASCII 差值**

**一图记四个**：

| 函数 | 干嘛 | 最常考 |
|------|------|--------|
| `strlen` | 测长度 | 不含 `\0`，别和 `sizeof` 搞混 |
| `strcpy` | 拷过去 | dest 要够大，不能是常量 |
| `strcat` | 接到尾 | dest 容量 = 两者之和 +1 |
| `strcmp` | 比大小 | 返回 **0** 才相等，不能用 `==` |

## 二、经典排序与查找算法（⭐必考编程题）

### 2.1 选择排序
```
每轮在未排序部分找最小值，放到已排序末尾
for (i = 0; i < n-1; i++)
    for (j = i+1; j < n; j++)
        if (a[i] > a[j]) 交换 a[i] 和 a[j]
```

### 2.2 冒泡排序
```
相邻两两比较，大的往后沉
for (i = 0; i < n-1; i++)
    for (j = 0; j < n-1-i; j++)
        if (a[j] > a[j+1]) 交换
```

### 2.3 折半查找（二分查找）
```
前提：数组已排序
每次取中间值，缩小一半范围
mid = (low + high) / 2
若 key == a[mid] → 找到
若 key <  a[mid] → high = mid-1
若 key >  a[mid] → low  = mid+1
```

### 2.4 筛选法求素数
```
1. 初始所有数为素数
2. 从2开始，将其倍数全部筛去
3. 剩余即为素数
```

## 三、高频考点（期末必考）

### 🔥 考点1：数组下标
- 从 0 开始，第 n-1 结束
- `int a[10];` 最大下标是 9

### 🔥 考点2：字符串结束符 `\0`
- `"Hello"` 实际占 6 字节
- 不预留 `\0` 空间导致 `%s` 输出乱码

### 🔥 考点3：排序算法
- 选择法和冒泡法必会一种
- 读程序写结果常考

### 🔥 考点4：杨辉三角形
```
a[i][0] = a[i][i] = 1
a[i][j] = a[i-1][j-1] + a[i-1][j]
```

## 四、易错点警示（⚠️ 坑点汇总）

| 易错点 | 说明 |
|--------|------|
| 下标越界 | `int a[10]; a[10]=5` 错误，最大下标是9 |
| 数组长度用变量 | 建议使用 `#define N 10` 定义常量 |
| 字符串忘记 `\0` | `strlen("Hello")` = 5，但存储占6字节 |
| gets 不安全 | 不检查越界，可能造成缓冲区溢出 |
| strcmp 返回值 | 不是 -1/0/1 ！是 ASCII 差值或正/负/零 |
| 数组名非变量 | `a++;` 错误，数组名是地址常量 |
| 二维数组列数不能省 | `int a[][4]` 正确，`int a[3][]` 错误 |
