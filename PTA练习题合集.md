## 第一次作业

### 题1：char 与 int 的 printf 输出（15分）

阅读以下程序：

```c
#include <stdio.h>

int main()
{
    char c1, c2;
    c1 = 97;        // 字符 a 的 ASCII 码
    c2 = 98;        // 字符 b 的 ASCII 码
    printf("c1=%c,c2=%c\n", c1, c2);
    printf("c1=%d,c2=%d\n", c1, c2);
    return 0;
}
```

**原程序输出**：

```
c1=a,c2=b
c1=97,c2=98
```

**问题**：如果将程序第 3 行改为 `int c1, c2;`，运行时第一个 printf 会输出什么？

**答案**：`c1=a,c2=b`

> 📌 考点：`%c` 按字符输出，int 和 char 在此题中效果相同。97 和 98 分别是 `'a'` 和 `'b'` 的 ASCII 码。

### 题2：scanf 格式输入（25分）

用下面 scanf 函数输入数据，使 `a=3, b=7, x=8.5, y=71.82, c1='A', c2='a'`，在键盘上如何输入？

```c
#include <stdio.h>

int main()
{
    int a, b;
    float x, y;
    char c1, c2;
    scanf("a=%db=%d", &a, &b);
    scanf("%f%e", &x, &y);
    scanf("%c%c", &c1, &c2);
    return 0;
}
```

**有以下三种正确输入方式**：

**方式1（分三行）**：

```
a=3b=7
8.5
71.82Aa
```

**方式2（分两行）**：

```
a=3b=7
8.5 71.82Aa
```

**方式3（分两行）**：

```
a=3b=7 8.5
71.82Aa
```

> 📌 考点：
> ① `scanf("a=%db=%d")` 中的 `a=` 和 `b=` 必须原样输入  
> ② `%f` 和 `%e` 自动跳过空白字符  
> ③ `%c` 不跳过空白字符，所以 `Aa` 必须紧跟在最后一个数字后面

### 题4：计算纳税额（编程题）

个人所得税计算，应纳税款的计算公式如下：

| 收入        | 税率               | 示例                                            |
| ----------- | ------------------ | ----------------------------------------------- |
| 收入 ≤ 1000 | 0%                 | 收入 900，纳税额 0                              |
| 收入 ≤ 2000 | 超过 1000 部分 5%  | 收入 1200，纳税额 200×5%=10                     |
| 收入 ≤ 3000 | 超过 2000 部分 10% | 收入 2200，纳税额 1000×5%+200×10%=70            |
| 收入 ≤ 6000 | 超过 3000 部分 15% | 收入 5200，纳税额 1000×5%+1000×10%+2200×15%=480 |
| 收入 > 6000 | 超过 6000 部分 20% | 收入 7200，纳税额 …+1200×20%=840                |

要求必须使用 **if-else 结构**。输出应纳税额和实际收入。

```c
#include <stdio.h>

int main(void)
{
    double income = 0.0;
    if (scanf("%lf", &income) != 1) {
        return 0;
    }

    double tax = 0.0;
    if (income <= 1000.0) {
        tax = 0.0;
    } else if (income <= 2000.0) {
        tax = (income - 1000.0) * 0.05;
    } else if (income <= 3000.0) {
        tax = 1000.0 * 0.05 + (income - 2000.0) * 0.10;
    } else if (income <= 6000.0) {
        tax = 1000.0 * 0.05 + 1000.0 * 0.10 + (income - 3000.0) * 0.15;
    } else {
        tax = 1000.0 * 0.05 + 1000.0 * 0.10 + 3000.0 * 0.15 + (income - 6000.0) * 0.20;
    }

    double afterTax = income - tax;
    printf("应纳税额: %.1lf, 实际收入: %.1lf", tax, afterTax);
    return 0;
}
```

> 📌 考点：分段函数的 if-else 多分支实现，累进税率逐级计算

### 题5：求三个数中的最大值（编程题）

从键盘输入 3 个整数，求其中的最大值并输出。

```c
#include <stdio.h>

int main(void)
{
    int a = 0, b = 0, c = 0;
    if (scanf("%d %d %d", &a, &b, &c) != 3) {
        return 0;
    }

    int max = a;
    if (b > max) {
        max = b;
    }
    if (c > max) {
        max = c;
    }

    printf("%d\n", max);
    return 0;
}
```

> 📌 考点：打擂台法求最值，先假设第一个为最大，再逐个比较更新
