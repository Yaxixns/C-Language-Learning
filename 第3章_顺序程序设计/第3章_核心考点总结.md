# 第3章 顺序程序设计 — 核心考点总结

---

## 一、本章核心知识点

### 1.1 数据的表现形式

#### 常量（⭐常考）
| 类型 | 示例 | 说明 |
|------|------|------|
| 整型常量 | `100`, `-5`, `0` | 不带小数点的整数 |
| 实型常量 | `3.14`, `-2.5`, `1.2e3` | 带小数点或指数形式 |
| 字符常量 | `'A'`, `'6'`, `'\n'` | 用单引号括起来的单个字符 |
| 字符串常量 | `"Hello"`, `"a"` | 用双引号括起来，自动加 `\0` |
| 符号常量 | `#define PI 3.1416` | 用标识符代表常量 |

**字符 'a' 与字符串 "a" 的区别**（⭐高频）：

| | `'a'` | `"a"` |
|------|------|------|
| 长度 | 1 字节 | 2 字节（含 `\0`） |
| 类型 | `char` | `char[]` |

#### 变量
- **定义**：程序运行期间值可以改变的量
- **先定义，后使用**
- **变量名**：字母/下划线开头，字母+数字+下划线组成，区分大小写

### 1.2 数据类型（⭐核心必考）

```
数据类型
├── 基本类型
│   ├── 整型 int（2或4字节，-32768~32767 或 -2^31~2^31-1）
│   ├── 字符型 char（1字节，-128~127）
│   ├── 单精度浮点型 float（4字节，6~7位有效数字）
│   └── 双精度浮点型 double（8字节，15~16位有效数字）
├── 构造类型（数组、结构体、共用体、枚举）
├── 指针类型
└── 空类型 void
```

### 1.3 运算符与表达式（⭐重点）

#### 算术运算符
| 运算符 | 含义 | 优先级 |
|--------|------|--------|
| `* / %` | 乘、除、取余 | 高 |
| `+ -` | 加、减 | 低 |

- `%` 两侧必须为**整型**
- 整数除整数结果仍为整数（截断小数）：`5/3 = 1`

#### 自增自减
| 形式 | 含义 |
|------|------|
| `++i` | 先加1，后使用 |
| `i++` | 先使用，后加1 |

#### 赋值运算符与类型转换
- `=` 赋值，`+=` `-=` `*=` `/=` `%=` 复合赋值
- **自动类型转换**：`int → float → double`（向高精度转）
- **强制类型转换**：`(类型名)表达式`，如 `(int)3.14`

### 1.4 输入输出函数（⭐必考）

#### printf — 格式输出
```c
printf("格式控制", 输出列表);
```

| 格式符 | 含义 | 格式符 | 含义 |
|--------|------|--------|------|
| `%d` | 有符号十进制整数 | `%f` | 单精度浮点数 |
| `%c` | 单个字符 | `%lf` | 双精度浮点数 |
| `%s` | 字符串 | `%e` | 指数形式 |
| `%o` | 八进制整数 | `%x` | 十六进制整数 |
| `%u` | 无符号十进制整数 | `%%` | 输出百分号本身 |

**附加格式**：
- `%5d` — 宽度5，右对齐
- `%.2f` — 保留2位小数
- `%-5d` — 左对齐

#### scanf — 格式输入
```c
scanf("格式控制", 地址列表);   // ⚠️ 变量前必须加 &
```

| 关键点 | 说明 |
|--------|------|
| `&` 地址符 | `scanf("%d", &a);` 必须加 `&` |
| 分隔符 | 空格、Tab、回车均可作为输入分隔 |
| 格式匹配 | `scanf("a=%d", &a)` 输入时必须写 `a=3` |

#### putchar / getchar — 字符输入输出
```c
char c;
c = getchar();    // 从键盘读一个字符
putchar(c);       // 输出一个字符
```

## 二、高频考点（期末必考）

### 🔥 考点1：printf 格式符
- **题型**：读程序写结果
- `printf("%c %d", 97, 'a');` → 输出 `a 97`

### 🔥 考点2：scanf 取地址符
- **易错**：`scanf("%d", a);` ❌ → `scanf("%d", &a);` ✅

### 🔥 考点3：char 与 int 的关系
- char 存储的是 ASCII 码值（整数），char 和 int 在一定范围内可互换
- `'A' = 65`, `'a' = 97`, `'0' = 48`
- char 范围 -128~127，超过127会溢出

### 🔥 考点4：整数除法截断
- `5/3 = 1`（不是1.666）
- `1/2 = 0`（整数除法，结果为0）

### 🔥 考点5：++i 和 i++ 的区别
- 单独的 `i++;` 和 `++i;` 等价
- 参与表达式时不同：`a = i++` vs `a = ++i`

## 三、易错点警示（⚠️ 坑点汇总）

| 易错点 | 错误 | 正确 |
|--------|------|------|
| scanf 忘写 & | `scanf("%d", a);` | `scanf("%d", &a);` |
| 取余用浮点数 | `5.0 % 2` | `%` 只能用于整数 |
| 整数除整数 | `5/3` 得 1 | `5.0/3` 才得 1.666... |
| 字符常量/字符串混淆 | `"A"` 当字符用 | 字符用 `'A'`，字符串用 `"A"` |
| scanf 格式多余字符 | `scanf("a=%d", &a)` | 输入时必须原样输入 `a=3` |
| char 溢出 | `c1=197` char 存不下 | char 范围 -128~127，改用 int |
| printf 打印 char 用 %d | 把字符当整数输出 | 输出 ASCII 码用 `%d`，输出字符用 `%c` |
