## 一、本章核心知识点

### 1.1 三种循环语句（⭐核心必考）

#### while 循环（先判断，后执行）
```c
while (条件表达式)
{
    循环体;
}
// 可能一次都不执行
```

#### do...while 循环（先执行，后判断）
```c
do
{
    循环体;
} while (条件表达式);
// ⚠️ 分号不能丢！至少执行一次
```

#### for 循环（最常用）
```c
for (表达式1; 表达式2; 表达式3)
{
    循环体;
}
// 表达式1：初始 → 表达式2：条件 → 循环体 → 表达式3：步进
```

| 对比 | while | do-while | for |
|------|-------|----------|-----|
| 最少执行次数 | 0次 | 1次 | 0次 |
| 适用场景 | 未知循环次数 | 至少执行一次 | 已知循环次数 |

### 1.2 循环控制语句

| 语句 | 作用 | 关键点 |
|------|------|--------|
| `break` | 跳出**当前**循环 | 只跳出一层 |
| `continue` | 跳过本次循环剩余部分，进入下一轮 | 不跳出循环 |

### 1.3 循环嵌套
- 外层循环一次，内层循环全部执行完
- 内层 break 只跳出内层，不影响外层

### 1.4 经典算法模板（⭐必考编程题）

#### ① 求最大公约数（辗转相除法 / 欧几里得算法）
```
while (n != 0)
{
    r = m % n;
    m = n;
    n = r;
}
// m 即为最大公约数
// 最小公倍数 = 原两数之积 / 最大公约数
```

#### ② 判断素数
```
若 n < 2，不是素数
循环 i 从 2 到 sqrt(n)：
    若 n % i == 0 → 不是素数
→ 是素数
```

#### ③ 求阶乘 n! / 累加求和
```c
// 阶乘
for(i=1, fac=1; i<=n; i++) fac *= i;
// 累加
for(i=1, sum=0; i<=n; i++) sum += i;
```

## 三、易错点警示（⚠️ 坑点汇总）

| 易错点 | 说明 |
|--------|------|
| do-while 后忘分号 | `while(...)` 后必须有 `;` |
| 循环变量未初始化 | 如 `int i; while(i<10)` 未赋初值 |
| 无限循环 | 循环条件永远为真，或忘记更新循环变量 |
| break vs continue | break 跳出循环，continue 跳过本轮 |
| 循环嵌套 break | 只跳出一层，不跳出外层 |
| 整数溢出 | 20! 用 int 存不下！改用 long long 或 double |
| for 分号 | `for(;;)` 是死循环，`for(;i<10;)` 正确但奇怪 |
| while 后多分号 | `while(x>0);` 死循环，分号是空语句 |
