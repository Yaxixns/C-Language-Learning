## 一、三种循环语句

| 循环 | 特点 | 最少执行 |
|------|------|---------|
| `while(条件)` | 先判断后执行 | 0次 |
| `do {...} while(条件);` | 先执行后判断，⚠️分号 | 1次 |
| `for(初;条;步)` | 三合一，最常用 | 0次 |

```c
// while — 适合未知次数
while (b != 0) { t = a % b; a = b; b = t; }

// do-while — 至少执行一次（如牛顿迭代）
do { x0 = x1; x1 = (x0 + a/x0) / 2; } while (fabs(x1 - x0) > 1e-5);

// for — 适合已知次数
for (i = 1; i <= 20; i++) { temp *= i; sum += temp; }
```

### break vs continue

| 语句 | 效果 |
|------|------|
| `break` | 跳出**当前层**循环 |
| `continue` | 跳过本次剩余，进入下一轮 |

## 二、本章练习错题复盘

### ⚠️ 5-4 统计字符 — fgets 陷阱
**错误**：`while(str[i]!='\0')` 把 `\n`（fgets保留的换行）计入了 other  
**修正**：`while(str[i]!='\0' && str[i]!='\n')`

### ⚠️ 5-7 三组求和 — 循环范围串用
**错误**：`if(i<=50) sum3+=1.0/i` 把 sum3（应只到10）绑到了 50  
**修正**：sum3 单独用 `if(i<=10)` 控制

### ⚠️ 5-11 球弹跳 — 变量含义混淆
**错误**：代码中 `h` 实际存的是 **2×反弹高度**，直接输出 h 当作反弹高度，多了一倍  
**修正**：输出 `h/2.0`

### 🔥 5-6 阶乘和 — 数据类型（自总结很到位）
- `int` 存不下 20!（溢出）→ 用 `unsigned long long`
- `%d` 打不了 → 用 `%llu`
- 循环内重复声明 `int i` → 编译错误

### 🔥 5-13 迭代法 — do-while vs while
- `while` + 假初值骗循环可能只跑一轮就退出
- 迭代类问题 `do-while` 更自然：先算再判断

## 三、经典题型速查

| 题型 | 核心思路 |
|------|---------|
| 最大公约数 | 辗转相除 `while(b!=0){t=a%b;a=b;b=t;}` |
| 水仙花数 | 三位拆分：百=i/100, 十=i/10%10, 个=i%10, 判立方和 |
| 完数 | 因子只查到 i/2，双遍扫描（先求和再输出） |
| 分数序列 | 分子分母是相邻斐波那契数：`next=a+b; b=a; a=next` |
| 猴子吃桃 | 倒推：`n = (n+1)*2`，从第10天倒推9次 |
| 牛顿迭代 | `x = x - f(x)/f'(x)`，`do-while` + `fabs` |
| 二分法 | 中点 `(L+R)/2`，f(mid)<0 → L=mid，否则 R=mid |
| 菱形图案 | 对称公式：`stars = n - 2*abs(i - mid)` |

## 四、本章易错清单

1. `fabs()` 用于浮点绝对值，`abs()` 用于整数 — 用错无报错但结果错
2. `do-while` 末尾分号不能丢
3. 整除陷阱：`4/3 = 1`，应该写 `4.0/3.0`
4. 循环变量不更新 → 死循环
5. `float` 精度有限，迭代类问题优先用 `double`
6. 数据太大用 `unsigned long long`（阶乘、幂次等）
