## 2. printf 格式说明符

格式：`%[标志][宽度][.精度]类型字符`

### 2.1 基本类型说明符

| 说明符 | 含义 | 示例 | 输出 |
|--------|------|------|------|
| `%d` | 有符号十进制整数 | `printf("%d", 97);` | `97` |
| `%u` | 无符号十进制整数 | `printf("%u", -1);` | `4294967295` |
| `%ld` | 长整型 | `printf("%ld", 100000L);` | `100000` |
| `%f` | 单双精度浮点数（小数形式） | `printf("%f", 3.14);` | `3.140000` |
| `%lf` | 双精度浮点数（printf中与%f等价） | `printf("%lf", 3.14);` | `3.140000` |
| `%e` | 科学计数法（小写e） | `printf("%e", 314.0);` | `3.140000e+02` |
| `%E` | 科学计数法（大写E） | `printf("%E", 314.0);` | `3.140000E+02` |
| `%g` | 自动选%f或%e（不输出多余零） | `printf("%g", 3.14);` | `3.14` |
| `%c` | 单个字符 | `printf("%c", 65);` | `A` |
| `%s` | 字符串 | `printf("%s", "abc");` | `abc` |
| `%o` | 无符号八进制 | `printf("%o", 10);` | `12` |
| `%x` | 无符号十六进制（小写） | `printf("%x", 255);` | `ff` |
| `%X` | 无符号十六进制（大写） | `printf("%X", 255);` | `FF` |
| `%p` | 指针（地址） | `printf("%p", &a);` | `0061FF1C` |
| `%%` | 输出百分号本身 | `printf("100%%");` | `100%` |

### 2.2 记忆口诀

```
d 整 f 浮 c 字符 s 串
o 八 x 十六 e 科学
u 无符 ld 长整 % 本身
```

## 3. printf 进阶格式控制

### 3.1 宽度控制

| 格式 | 含义 | 示例 `printf` | 输出 |
|------|------|-------------|------|
| `%5d` | 至少占5列，右对齐 | `printf("%5d", 42);` | `   42` |
| `%-5d` | 至少占5列，左对齐 | `printf("%-5d", 42);` | `42   ` |
| `%05d` | 不足5列用0填充 | `printf("%05d", 42);` | `00042` |

### 3.2 精度控制（浮点数）

| 格式 | 含义 | 示例 | 输出 |
|------|------|------|------|
| `%.2f` | 保留2位小数 | `printf("%.2f", 3.14159);` | `3.14` |
| `%.0f` | 不保留小数 | `printf("%.0f", 3.6);` | `4`（四舍五入） |
| `%8.2f` | 总宽8列，2位小数 | `printf("%8.2f", 3.14);` | `    3.14` |
| `%10.3f` | 总宽10列，3位小数 | `printf("%10.3f", 1.5);` | `     1.500` |

### 3.3 字符串截断

| 格式 | 含义 | 示例 | 输出 |
|------|------|------|------|
| `%.3s` | 只输出前3个字符 | `printf("%.3s", "Hello");` | `Hel` |
| `%8.3s` | 8列宽，只显示前3字符 | `printf("%8.3s", "Hello");` | `     Hel` |

## 5. scanf 格式说明符

### 5.1 基本用法

```c
scanf("格式串", &变量1, &变量2, ...);
//                     ↑ 注意：必须加取地址符 &
```

| 说明符 | 含义 | 示例 |
|--------|------|------|
| `%d` | 读入十进制整数 | `scanf("%d", &a);` |
| `%ld` | 读入长整型 | `scanf("%ld", &a);` |
| `%f` | 读入 float 浮点数 | `scanf("%f", &x);` |
| `%lf` | 读入 double 浮点数 | `scanf("%lf", &x);` |
| `%c` | 读入一个字符（⚠️含空格） | `scanf("%c", &c);` |
| `%s` | 读入字符串（到空白为止） | `scanf("%s", str);` |

### 5.2 格式串中的普通字符

```c
scanf("a=%d,b=%d", &a, &b);
//     ^^  ^^  这些普通字符必须原样输入！
// 正确输入：a=3,b=7
// 错误输入：3 7  ← 缺少 a= 和 ,
```

### 5.3 连续输入

```c
// 数值连续输入 — 空格/回车/Tab 自动分割
scanf("%d%d%d", &a, &b, &c);  // 输入：3 7 5 或 3换行7换行5

// 字符连续输入 — 无空白分割
scanf("%c%c", &c1, &c2);      // 输入：AB （中间不能有空格！）
```

## 6. scanf 的常见陷阱

### 6.1 陷阱一：%c 不跳过空白字符

> **这是考试最高频的坑！**

```c
scanf("%d", &n);
scanf("%c", &ch);
// 输入：123换行A
// 结果：n=123, ch='\n' （读到了换行符，不是'A'！）
```

**解决方案：**

```c
// 方案1：在 %c 前加空格，让 scanf 跳过所有空白
scanf(" %c", &ch);  // 注意 %c 前面有个空格！

// 方案2：用 getchar() 吃掉缓冲区残留
scanf("%d", &n);
getchar();          // 吃掉换行符
scanf("%c", &ch);
```

### 6.2 陷阱二：格式串中的普通字符必须原样输入

```c
scanf("x=%f,y=%f", &x, &y);
// 必须输入：x=8.5,y=71.82  ← 字母和逗号一个不能少
```

### 6.3 陷阱三：scanf 到 %s 不读空格

```c
char str[20];
scanf("%s", str);
// 输入：Hello World
// 结果：str = "Hello" （World 留在缓冲区）
```

### 6.4 陷阱四：浮点类型混淆

```c
float  x;   scanf("%f", &x);   // 正确：用 %f
double y;   scanf("%lf", &y);  // 正确：用 %lf  ← 区分！
```

```c
// 但 printf 中 %f 和 %lf 通用
printf("%f", x);   // 正确
printf("%f", y);   // 正确（printf对float/double都用%f）
```

## 7. 字符输入输出

### 7.1 putchar

```c
putchar('A');        // 输出：A
putchar(65);         // 输出：A （ASCII码65='A'）
putchar('\n');       // 输出换行
putchar(c1);         // 输出变量 c1 对应的字符
```

### 7.2 getchar

```c
char c;
c = getchar();       // 从键盘读入一个字符（含空格/换行）
```

**典型用途：吃换行符**
```c
scanf("%d", &n);
getchar();           // 吃掉 scanf 遗留的换行符
c = getchar();       // 现在读到的是真正的字符
```

### 7.3 字符与 ASCII 码互转

```c
char c = 'A';
printf("%c", c);     // 按字符输出 → A
printf("%d", c);     // 按ASCII码输出 → 65

int n = 97;
printf("%c", n);     // 按字符输出 → a
printf("%d", n);     // 按整数输出 → 97
```

## 8. 字符串输入输出

### 8.1 puts — 输出字符串

```c
char str[] = "Hello";
puts(str);           // 输出 Hello 并自动换行
puts("World");       // 输出 World 并自动换行
```

### 8.2 gets — 读取一行（❌ 考试可能还在考，但已淘汰）

```c
char str[50];
gets(str);           // 读到换行符，丢弃换行，\0 结尾
// ⚠️ 危险：不检查数组长度，可能越界！
```

### 8.3 fgets — 安全替代

```c
fgets(str, 50, stdin);  // 最多读49字符，保留\0位置
// ⚠️ 注意：fgets 会保留换行符 \n 在字符串末尾
```

## 9. 标准易错点汇总

| 易错点 | 错误写法 | 正确写法 |
|--------|---------|---------|
| scanf 缺 & | `scanf("%d", a);` | `scanf("%d", &a);` |
| printf 加 & | `printf("%d", &a);` | `printf("%d", a);` |
| %c 吃空格 | `scanf("%c%c", &c1, &c2);` 输入`A B` | `scanf(" %c %c", &c1, &c2);` |
| 浮点输入用错 | `scanf("%d", &x);` | `scanf("%f", &x);` |
| printf 精度误用 | `printf("%.2d", 5);` | `printf("%.2f", 3.14);`（精度只对浮点数有意义） |
| double 输入 | `scanf("%f", &y);` | `scanf("%lf", &y);` |
| 格式串有普通字符 | 只输入数据不输入普通字符 | 必须原样输入格式串中的普通字符 |

## 10. 期末考试常见题型

### 题型一：写输出结果

```c
int a = 5, b = 7;
printf("a=%d,b=%d\n", a, b);
// 输出：a=5,b=7
```

### 题型二：字符与ASCII互转

```c
char c = 'A';
printf("%c %d\n", c, c);
// 输出：A 65
```

### 题型三：格式控制

```c
printf("%5d\n", 123);    // 输出：  123
printf("%05d\n", 123);   // 输出：00123
printf("%.2f\n", 3.1);   // 输出：3.10
printf("%-5d\n", 123);   // 输出：123
```

### 题型四：scanf 陷阱分析

```c
int n; char c;
scanf("%d", &n);
scanf("%c", &c);
// 输入：10换行A
// 问：c 读到了什么？
// 答：'\n'（换行符），不是 'A'
```

### 题型五：格式串中的普通字符

```c
scanf("a=%db=%d", &a, &b);
// 应如何输入使 a=3, b=7？
// 答：a=3b=7
```

### 题型六：char 溢出

```c
char c = 197;           // 超出 signed char 范围
printf("%d\n", c);
// 输出：-59 （发生溢出）
```

### 题型七：short-circuit（短路求值）

```c
int a = 3, b = 4;
// !(a > b) && !(c = 5) && 0  
// 问：这一步执行后 c 的值是？
// 答：c 未被赋值，因为 && 短路 —— 左边为假，右边不执行
```
