# 第1章 程序设计和C语言 — 核心考点总结

---

## 一、本章核心知识点

### 1.1 程序与程序设计
| 概念 | 定义 |
|------|------|
| **程序** | 计算机能够识别和执行的一组指令（有序的指令集合） |
| **程序设计** | 设计、编制、调试程序的过程 |
| **程序设计语言** | 人与计算机交流的规则化语言 |

### 1.2 计算机语言的发展
```
机器语言 → 汇编语言 → 高级语言
（二进制）  （助记符）   （接近自然语言）
```
- **机器语言**：计算机唯一可直接执行的语言，但难以记忆和使用
- **汇编语言**：用助记符代替二进制，需汇编程序翻译
- **高级语言**：接近人类自然语言和数学语言，需编译程序翻译，可移植性好

### 1.3 C语言的发展与特点
- **诞生**：1972年，贝尔实验室，Dennis Ritchie 设计
- **ANSI C 标准**：1983年制定，1989年通过（ANSI X3.159-1989）
- **ISO C 标准**：1990年，国际标准化组织采纳（ISO/IEC 9899:1990）
- **C99/C11/C17**：后续标准更新

**C语言的主要特点**（⭐常考简答）：
1. 语言简洁、紧凑，使用方便灵活（仅37个关键字）
2. 运算符丰富（34种运算符）
3. 数据类型丰富（整型、浮点型、字符型、指针、结构体等）
4. 具有结构化的控制语句
5. 语法限制不太严格，程序设计自由度大
6. 允许直接访问物理地址，能进行位操作
7. 生成目标代码质量高，程序执行效率高
8. 可移植性好

### 1.4 简单C语言程序的结构（⭐核心必考）

```c
#include <stdio.h>           // 编译预处理指令（头文件）

int main()                   // 主函数（程序入口）
{
    printf("Hello World\n"); // 函数体（语句）
    return 0;                // 返回值
}
```

**程序结构要点**：
1. 一个C程序**有且仅有一个** `main` 函数
2. 程序从 `main` 函数开始执行，`main` 函数执行完毕程序结束
3. `#include` 是**编译预处理指令**，不是语句（不用加分号）
4. 每个语句以 `;` 结束
5. `/* ... */` 为块注释，`//` 为行注释
6. C语言本身不提供输入输出语句，由库函数（如 `printf`、`scanf`）实现

### 1.5 C程序的开发流程（⭐常考填空）
```
编辑(.c源文件) → 编译(.obj目标文件) → 连接(.exe可执行文件) → 运行
```

| 阶段 | 产物 | 说明 |
|------|------|------|
| 编辑 | `.c` 源文件 | 编写C源代码 |
| 编译 | `.obj` 目标文件 | 编译器将源程序翻译成机器指令 |
| 连接 | `.exe` 可执行文件 | 将目标文件与库函数等连接装配 |
| 运行 | 输出结果 | 执行程序 |

## 二、高频考点（期末必考）

### 🔥 考点1：C程序的基本结构
- **题型**：选择题 / 填空题
- **典型考法**：判断以下说法是否正确
  - "一个C程序可以有多个main函数" — ❌ 只能有一个
  - "C程序从第一个函数开始执行" — ❌ 从main函数开始
  - "分号是语句的结束标志" — ✅
  - "预处理指令必须以 `#` 开头" — ✅

### 🔥 考点2：程序开发流程顺序
- **题型**：排序题 / 选择题
- **口诀**：**编-译-连-运**（编辑→编译→连接→运行）

### 🔥 考点3：头文件与预处理
- `#include <stdio.h>` 的作用：包含标准输入输出库函数声明
- 不写 `#include` 能否使用 `printf`？理论上某些编译器会警告但可能通过，标准要求必须包含

### 🔥 考点4：C语言的特点
- **题型**：简答题 / 选择题
- 常考：C语言属于**高级语言**，但兼具低级语言功能（可直接访问地址）

## 三、易错点警示（⚠️ 坑点汇总）

| 易错点 | 错误示例 | 正确做法 |
|--------|----------|----------|
| 预处理指令加分号 | `#include <stdio.h>;` | `#include <stdio.h>` |
| main函数拼写错误 | `mian()` / `Main()` | `main()` |
| 忘记返回值 | `void main()` | `int main()` 并 `return 0;` |
| 中文符号 | `printf（"Hello"）;` | `printf("Hello");`（全英文符号） |
| 注释嵌套 | `/* a /* b */ c */` | C语言注释不能嵌套 |
| 漏写分号 | `printf("Hello")` | `printf("Hello");` |
