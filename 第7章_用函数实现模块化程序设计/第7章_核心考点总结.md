## 一、函数核心知识

### 1.1 函数定义

```c
返回类型 函数名(参数列表)    // 没有返回值用 void
{
    函数体;
    return 值;              // void 函数可省略 return
}
```

### 1.2 函数声明 vs 定义

```c
// 声明（告诉编译器有这个函数）— 放 main 之前或头文件
int gcd(int a, int b);

// 定义（函数体）— 可以放 main 之后
int gcd(int a, int b) { ... }
```

### 1.3 参数传递

```c
void swap(int x, int y) { ... }   // 值传递：不影响实参
void swap(int *p1, int *p2) { ... } // 地址传递：影响实参（第8章详细讲）

// 数组作参数（传递的是地址）
void sort(int a[], int n) { ... }  // 修改数组会影响实参！
```

### 1.4 局部变量 vs 全局变量

| | 局部变量 | 全局变量 |
|------|---------|---------|
| 作用域 | 函数内部 | 整个文件 |
| 生命周期 | 函数执行期间 | 程序运行全程 |
| 可读性 | ✅ 好 | ⚠️ 破坏模块化 |

## 二、本章错题复盘

### ⚠️ 7-2 一元二次方程 — delta<0 处理
**错误**：`printf("无解\n")`  
**修正**：delta<0 有两共轭复根，应输出
```c
float real = -b/(2*a);
float imag = sqrt(-dt)/(2*a);
printf("x1=%.3f+%.3fi, x2=%.3f-%.3fi\n", real, imag, real, imag);
```

### ⚠️ 7-6 字符串连接 — 局部数组陷阱
**错误**：函数内部定义 `char str3[50]`，拼接后在函数内 printf，调用方拿不到结果  
**修正**：传入结果数组作为参数 `void connect(char s1[],char s2[],char result[])`

### ⚠️ 7-8 四位数字输出 — 尾随空格
**错误**：`printf("%d ", ...)` 每个数字后一个空格，最后多一个  
**修正**：`printf("%d %d %d %d\n", a,b,c,d)` 一次输出

### 🔥 7-10 最长单词 — 内容未保存
**错误**：只记录了最长长度，没保存最长单词的内容；最后一个单词（无空格结尾）没比较  
**修正**：记录最长单词的起始位置 `max_start`，输出时从该位置取 `max_len` 个字符

## 三、递归

```c
// 勒让德多项式 (7-13)
float p(int n, float x) {
    if (n == 0) return 1;
    if (n == 1) return x;
    return ((2*n-1)*x*p(n-1,x) - (n-1)*p(n-2,x)) / n;
}

// 整数转字符串 (7-17) — 先递归再输出 = 正序
void convert(int n) {
    if (n / 10) convert(n / 10);
    putchar(n % 10 + '0');
}
```

**递归要点**：①基案（终止条件）②递归公式 ③回溯时执行 = 逆序效果反转

## 四、函数指针（进阶）

```c
// 7-13：矩形法求定积分 — 传什么函数就算什么积分
double integral(double (*f)(double), double a, double b, int n) {
    double h = (b-a)/n, sum = 0;
    for (int i = 0; i < n; i++)
        sum += f(a + i*h) * h;       // 通用公式
    return sum;
}
integral(sin, 0, 1, 100000);  // ∫sin
integral(cos, 0, 1, 100000);  // ∫cos
```

## 六、易错清单

1. 函数内定义的局部数组，返回后内存被回收（别返回局部数组地址）
2. 数组作参数不需 `&`，函数内修改会影响原数组
3. 递归忘记终止条件 → 栈溢出
4. `void` 函数不需要 `return` 语句
5. 全局变量破坏模块化，尽量用参数传递
6. 函数声明末尾要分号，定义末尾不要分号
